# Shark Fish Challenge Game

A simple and fun web game where you control a shark to eat fish and avoid boats!

## How to Play

1. Open `index.html` in a web browser
   - **Note**: Due to CORS restrictions, you may need to run a local server:
     - Using Python: `python -m http.server 8000` then visit `http://localhost:8000`
     - Using Node.js: `npx http-server` then visit the provided URL
     - Or use any local web server

2. **Controls**:
   - Use **Arrow Keys** or **WASD** to move the shark
   - Eat the golden fish (üêü) to increase your score
   - Avoid the brown boats (üö§) or it's game over!

3. **Gameplay**:
   - Score increases each time you eat a fish
   - Difficulty increases every 10 seconds
   - More boats and faster fish spawn as difficulty increases
   - Try to survive as long as possible and get the highest score!

## Features

- 3D shark model (shark.glb)
- Smooth controls and camera following
- Increasing difficulty over time
- Score tracking
- Game over screen with final score
- Minimalistic yet fun design

Enjoy the game!
